CSV Format Requirements

Your CSV file must contain the following columns (case-sensitive):

Column Name Description Example
slno Serial number (optional, can be empty) 1, 2, 3...
question The question text Which of the following is a Python keyword?
option_a Option A text variable
option_b Option B text if
option_c Option C text number
option_d Option D text string
correct_answer Correct answer (A, B, C, or D) B
Important Notes:
  • Column names are case-sensitive and must match exactly
  • Correct answer must be A, B, C, or D (uppercase)
  • All text fields should be properly escaped if they contain commas or quotes
  • The system will assign the selected module number to all questions in the CSV
